## 💻 실행 방법
프로젝트 실행을 위한 환경 및 명령어를 안내합니다.

### 1) 레포지토리 클론
    git clone https://github.com/사용자/레포.git

### 2) 프로젝트 폴더 이동
    cd 프로젝트-폴더명

### 3) 패키지 설치
    npm install
    # 또는
    yarn install

### 4) 실행
    npm start
    # 또는
    yarn start

**추가 안내:**  
- Node.js 요구 버전  
- 환경변수(.env) 설정 방법  
- 빌드 방법 등
